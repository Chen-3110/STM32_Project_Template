# STM32F407 CNC Plotter 输入输出规范

## 1. 输入参数
### 1.1 串口指令格式
- 格式：`X[值] Y[值] Z[值] F[值]`
- 示例：`X10.5 Y20.3 Z5.0 F6000`
- 参数说明：
  - X：X轴目标位置（单位：毫米）
  - Y：Y轴目标位置（单位：毫米）
  - Z：Z轴目标位置（单位：毫米）
  - F：运动速度（单位：Hz，范园400-12000）
- 协议：停-等机制，每行以换行符(\n)结束

### 1.2 限位开关
- 引脚分配：
  - X轴限位：PE7（低电平触发）
  - Y轴限位：PE8（低电平触发）
  - Z轴限位：PE9（低电平触发）

### 1.3 步进电机参数
- 每毫米步数：
  - X轴：200 steps/mm
  - Y轴：200 steps/mm
  - Z轴：400 steps/mm

## 2. 输出参数
### 2.1 步进电机控制
- X轴：
  - 步进脉冲：PB10
  - 方向控制：PB11
- Y轴：
  - 步进脉冲：PB12
  - 方向控制：PB13
- Z轴：
  - 步进脉冲：PB0 (TIM3_CH3 PWM)
  - 方向控制：PB1

### 2.2 状态反馈
- 运动完成后通过UART1发送：`OK\n`
- 错误状态：未实现

### 2.3 电机使能
- 使能引脚：PB2（低电平有效）

## 3. 坐标系
- 当前坐标系：绝对坐标系
- 原点位置：未定义（取决于硬件设置）

## 4. 运动参数
- 默认速度：5000 Hz
- 速度范园：400-12000 Hz
- 加减速类型：T型加减速
- 加减速曲线：内部算法控制
